### Paso 1
```
- ingresar al link y descargar nodejs
https://nodejs.org/es/download/

- descargar mongodb localmente 
```

### Paso 2
```
- despues de hacer el paso 1 instalar globalmente ionic y cordova  con el siguinete comando 

es recomendable ejecutar en modo administrado si tiene sistema operativo w7 o w10

$ npm i -g ionic cordova 
```

### Paso 3 
```
- ir al repositoria y clonar el proyecto 
- despues de haber clonado el proyecto con el siguiente comando instalar las dependencias 

$ npm i 

```

### Paso 4

```
cuando quiere subir cambios al repositorio verificar que cambios hicieron y luego subirlo con los siguientes comando

- es necesario tener git instalado 

1) git status - verifica que archivo modifico 
2) git add . 
3) git commit -m "comentario"
4) git push origin nombreRama

```
